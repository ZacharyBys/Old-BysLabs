Avalon

Android Java

Avalon Won 2nd place at Blochacks 2017. It solves the problem of processing and keeping track of refugee information. On the client app, users can register to update their information, and send alerts to a Refugee Center. On the Admin App, The Centers have access to the information, a map, and receive notifications from refugees.

Hashtag Game

Android Java

This was my first Android application, made using Android Studio. The app is a game in which two Instagram hashtags appear, and the player must guess which one has been used more. The data comes from the Instagram app. It is available on the Google play store, and contains ads.

ShotBot

Arduino C++

ShotBot won Best Electrical Robot at the Mcgill Hardware hackathon "RoboHacks 2017". Our team of 4 designed and built an automatic drink dispenser, which was activated from a mobile device. ShotBot was able to pour drinks precisely into the glass and skip hole that did not have a glass using infrared sensors.
